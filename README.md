# smart-file-organizer
Smart File Organizer is a python script that organizes files into folders based on their file types

# Features
* Sorts files into categories:
  * Audio
  * Video
  * Images
  * Documents
  * Others

* Easy to use
* Beginer friendly

# How it works
* The script scans files in the selected folder
* detects file type using file extensions
* Moves each file into it's respective folder
